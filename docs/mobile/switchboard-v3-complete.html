The loopFive moves. The AI does four of them.

Switchboard inverts the PM tool: the agent creates, schedules, and ships; the app is where you watch and steer. The whole product is one loop, and the phone runs all of it.

01

Kickoff

Describe a goal. The plan authors itself — epics, tasks, deps, owners.

05

Supervise

Approve, redirect, or veto. It pings you only when it needs a human.

What ships today

live Kickoff intake, dispatch (claim_next), live narration, redirect/stop (revoke_claim), deliverables & mission_status, dependency map, closure gates, file leases.

Thin adds

The roadmap

build The full autopilot loop (PR→merge→auto-dispatch) and push (notify + APNs) — the two builds that turn the on-ramp into the destination.

Design system

Taikun red #c0392b as accent + "live"; native SF Pro UI with SF Mono for the technical layer; dynamic-island frames, a 5-tab bar, swipe-approve, live pulse dots. On-brand with taikun-tabler.css.
