Design systemBuilt on the real Switchboard, not a mockup.

Palette

Taikun red #c0392b is the only accent — it also signals "live." Semantic greens, ambers and slates are kept separate from the brand and used only for state. The cockpit inverts to near-black with red telemetry glow.

Typography

SF Pro (native system) carries the UI — authentic on-device and CSP-safe, no webfont. SF Mono voices the technical layer: task IDs, session handles, spend, timestamps. Display weights for the numbers you watch move.

Native patterns

Dynamic-island frame, 9:41 status bar, a 5-tab bar with a raised center Kickoff action, swipe-approve cards, live pulse dots, and a home indicator. Motion is limited to state that's genuinely changing.

Maps to the control plane

Kickoff → intake.py · goal→plan
Autopilot → dispatch.py loop
Session → narration + interrupts
Deliverable → evidence gates
Approvals → action queue · Tally

Switchboard mobile · the AI operates, you supervise. Wireframes are directional — every surface traces to a shipping module.
