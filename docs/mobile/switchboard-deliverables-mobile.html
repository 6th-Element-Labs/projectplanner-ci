Every active agent broadcasts a soft lease on the files it's editing. The view warns before two agents touch the same file, and when main moves under an open branch — the exact coordination pain that eats a multi-agent day, surfaced as one signal.

Waves come straight from the dependency graph: Wave 1 = the ready/unblocked set (claim_next eligibility), later waves gated by keystone merges. peek_next previews the batch without claiming it.

Honest by construction

Done means merged with terminal provenance. A Done task lacking proof is "Done · no proof"; closure grades PASS / WAIVE / HOLD and a required gate with no result reads NOT RUN. The brief is derived from status, not optimism.
