### 2.6 Darko's caveat — "no API gateway yet, be creative"

We have a **3-option bootstrap plan** for the period while the gateway is built:

1. **S3 JSON drops** (recommended) — TEEP places scheduled JSON in an S3 bucket matching the OpenAPI response shapes. We poll signed URLs.
2. **SFTP / managed file transfer** — same pattern, different transport.
3. **Direct read of a TEEP read-replica** with views — only as a temporary bridge.

Sierra's xlsx is effectively a Bootstrap-0 mode for Sensirion data already. Extending to the other 4 systems unblocks Day-1 dev.

See [04-system-integrations.md §6](04-system-integrations.md).
